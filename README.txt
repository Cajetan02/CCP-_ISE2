
# Data Visualizations Project

This project includes Python classes and methods for various types of data visualizations. 
The datasets are provided as CSV files.

## Visualizations Included
1. Bar Chart of Sales Data
2. Line Chart of Temperature Changes
3. Scatter Plot of Student Scores
4. Histogram of Exam Scores
5. Heatmap of Correlation
6. Wave Plot of Sine and Cosine
7. Box Plot of House Prices

## Usage
1. Install required Python libraries: `matplotlib`, `pandas`, `seaborn`, `numpy`.
2. Import the corresponding class from `visualizations.py`.
3. Use the appropriate method with the given dataset path.
